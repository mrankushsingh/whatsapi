                            Quick Start
                        

                        
Make a POST request to the /scrape endpoint:

                        
curl -X POST {{ request.url_root }}scrape \
  -H "Content-Type: application/json" \
  -d '{
    "link": "https://chat.whatsapp.com/XXXXXXXXX"
  }'
